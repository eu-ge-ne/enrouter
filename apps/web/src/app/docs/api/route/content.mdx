# Route API

## Interfaces

### Route

```typescript
export interface Route {
  /**
   * Route's path
   * @see https://github.com/lukeed/regexparam
   */
  path: string;

  mod: string[];

  /**
   * Urls of assets associated with the route
   */
  link: [string[], string[]]; // css, modules

  tree?: Route[];
}
```
