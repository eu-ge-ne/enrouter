# Getting Started

Will soon be updated...

## Install

```bash
$ pnpm install enrouter
```

## Build route modules

```ts
import { buildRouteModulesFromViteGlobs } from "enrouter";

const modules = buildRouteModulesFromViteGlobs({
  globs: import.meta.glob(["./app/**/_*.tsx"]),
  moduleId: (key) => "src" + key.slice(".".length),
  path: (key) => key.slice("./app/".length),
});
```

## Build module assets

```ts
import { buildModuleAssetsFromViteManifest } from "enrouter";

import manifest from "@app/web/manifest";

const assets = buildModuleAssetsFromViteManifest({
  manifest,
  toUrl: (x) => new URL(x, "http://localhost").pathname,
});
```
